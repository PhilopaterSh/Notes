# SEC-101: Foundations of Cybersecurity National Telecommunication Institute (NTI)  Cairo, Egypt  - 2025

Course Overview

This  course provides  a comprehensive  foundation  in cybersecurity,  combining  theoretical principles with practical  skills.  Students  will be prepared  for advanced  study  and industry certifications  by exploring core topics such as networking  fundamentals, operating  system security,  cryptography,  and  network  security  concepts.    The  course  follows a  structured approach:

Credit: 5 hours

Understand → Protect → Detect → Respond → Improve.

Learning Objectives

Upon successful completion of this course, students will be able to:

• Identify the fundamental  components of modern computers and the history of the internet
• Define core cybersecurity  concepts,  including  the  CIA Triad  (Confidentiality, Integrity, Availability)  and defense-in-depth  strategies
• Differentiate between various types of cyberattacks such as DoS/DDoS, phishing, and SQL
injection
• Recognize common forms of malware, including viruses, worms, Trojans,  and ransomware
• Analyze real-world security breaches to understand attack  vectors and defense failures
• Apply best practices for securing computers,  managing passwords, and protecting  wireless networks
• Explain the purpose of major cybersecurity  standards like ISO 27001 and PCI DSS
• Describe the basic principles of symmetric and asymmetric  cryptography

# Chapter 2: Cybersecurity Fundamentals
Protecting Digital Assets in a Connected World

## 2.1 Introduction to Cybersecurity
#Definition Cybersecurity involves protecting systems, networks, and programs from digital attacks.

- Protects sensitive data 
- Defends against unauthorized access 
- Ensures privacy and confidentiality 
- Maintains system integrity

Cybersecurity Domains
- Multiple interconnected domains 
- Each requires specialized approaches
- Balanced security across all areas 

## 2.2 Importance of Cybersecurity
Cybersecurity is critical in our increasingly digital world

**==Increasing Digital Dependency → Growing Cybersecurity Importance==**

*95% of cybersecurity breaches are due to **human error** (IBM)*

## 2.3 Introduction to Cyberattacks

What are Cyberattacks? 
	Deliberate exploitation of computer systems, networks, and technology-dependent enterprises.

Types of Cyber Attackers:

| External Threats | Internal Threats                |
| ---------------- | ------------------------------- |
| Hackers          | Disgruntled employees المستاؤون |
| Organized crime  | Careless staff                  |
| Nation-states    | Privileged users                |
| Hacktivists      | Former employees السابقون       |
| Script kiddies   | Third-party vendors             |

Attack Motivation
Cybersecurity is a game of strategy between attackers and defenders

## 2.4 Objectives of Cyberattacks
Cyberattacks typically target one or more security objectives

### CIA Triad: Foundation of Security
#Confidentiality: Preventing unauthorized disclosure. 
#Integrity: Preventing unauthorized modification. 
#Availability: Ensuring timely access to systems and data.

## 2.4.1 Confidentiality Breach
#Confidentiality Ensuring that information is not disclosed to unauthorized individuals, entities, or processes.

| Attack Examples      | Protection Methods |
| -------------------- | ------------------ |
| Eavesdropping التنصت | Encryption         |
| Data theft           | Access controls    |
| Email interception   | Authentication     |
| Database breaches    | Data masking       |
## 2.4.2 Availability Breach
#Availability Ensuring that systems and data are accessible and usable when needed by authorized users.

| Attack Examples         | Protection Methods |
| ----------------------- | ------------------ |
| Denial of Service (DoS) | Redundancy         |
| System crashes          | Backups            |
| Ransomware              | Load balancing     |
| Service disruptions     | DDoS protection    |
Requires specialized mitigation services

Ransomware: Holding Data Hostage
	How It Works 
	- Malware is installed 
	- Files are encrypted 
	- Ransom is demanded 
	- Decryption after payment
	Protection Keep 
	 - Keep regular backups 
	 - Train and educate users 
	 - Use network segmentation 
	 - Deploy anti-malware tools

## 2.4.3 Integrity Breach
#Integrity Maintaining the accuracy and completeness of data and systems.

| Attack Examples   | Protection Methods |
| ----------------- | ------------------ |
| Data manipulation | Hashing            |
| Code injection    | Digital signatures |
| Malware infection | Checksums          |
| Log tampering     | Audit logs         |
Data Integrity Techniques:
- Multiple techniques provide layered protection.
- Different methods appropriate for different scenarios.
- Balance between security and performance.

### Beyond the CIA Triad
Modern cybersecurity extends beyond the traditional CIA model.

| **Authentication**   | **Non-Repudiation**           | **Accountability** |
| -------------------- | ----------------------------- | ------------------ |
| Verifying identity   | Proof of actions دليل الأفعال | Action tracking    |
| Multi-factor methods | Digital signatures            | Audit trails       |
| Biometrics           | Legal evidence دليل           | Forensic analysis  |

### Authentication Methods

- Something You Are

### Types of Malware
- Virus 
	  Spyware
- Worm 
	  Adware 
- Trojan 
	  Rootkit
-  Ransomware 
	  Botnet

> Malicious software designed to damage or disrupt systems.
> Constantly evolving with new variants.
> Requires multi-layered defense strategies.
> User education is critical for prevention.

### Defense in Depth Strategy
1. Physical Security 
	Access controls, surveillance 
2. Network Security 
	Firewalls, IDS/IPS
3. Host Security 
    Antivirus, hardening 
4. Application Security 
   Secure coding, WAF 
5. Data Security
   Encryption, DLP

### Types of Security Controls
1. Preventive 
	- Firewalls 
	- Encryption 
	- Access controls 
	- Antivirus 
2. Detective 
	- IDS/IPS 
	- Alarms 
	- Audit logs 
	- CCTV 
3. Corrective 
	- Backups 
	- Redundancy 
	- Patches 
	- Incident response
